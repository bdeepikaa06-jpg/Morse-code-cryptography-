# 🔐 Morse Code Cryptography with Audio

This project converts text into Morse code, encrypts it using AES, and generates audio output.

---

## 🚀 Features
- Text to Morse conversion
- AES encryption & decryption
- Password protection
- Morse to text conversion
- Audio generation (.wav file)

---

## 🛠️ Technologies Used
- Python
- NumPy
- SciPy
- PyCryptodome

---

## ▶️ How to Run
pip install numpy scipy pycryptodome  
python morse_crypto.py  

---

## 📁 Output
- Decrypted text in terminal  
- Audio file: decoded_morse_code.wav
